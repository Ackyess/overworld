# NASA LRO Moon

- Source page: https://svs.gsfc.nasa.gov/14959/
- Online asset: `moon-2k.glb`
- Original asset: `Moon_NASA_LRO_8k_Topo_Small.glb`
- Data: Lunar Reconnaissance Orbiter imagery and topography
- Credit: NASA's Goddard Space Flight Center

The online scene keeps the original topographic mesh, limits the color map to
2048×1024, and uses WebP plus Draco for transfer. The untouched 8K input is in
the local source archive.
