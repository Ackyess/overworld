# Orbit world sources

## International Space Station

- Local file: `iss.glb`
- Source: https://science.nasa.gov/resource/international-space-station-3d-model/
- Credit: NASA Visualization Technology Applications and Development (VTAD)
- Original download: glTF model, 42.43 MB

Review NASA's media usage guidelines before public redistribution:
https://www.nasa.gov/nasa-brand-center/images-and-media/

## Earth maps

The four files in `earth/` are the planet textures distributed with the
Three.js examples:

https://github.com/mrdoob/three.js/tree/dev/examples/textures/planets

Three.js is distributed under the MIT license. Preserve the original filenames
because the runtime loads each map explicitly.
